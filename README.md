# Securian Nova Project

- Build a point-of-sale with back-end integration for the Advisor/Lender user to be used during the sales journey This will include functionality that begins at the "discovery" phase when CU lender or member initiates the conversation of creditor protection insurance to the "application" stage when the application for insurance is submitted.

# Start Build and Test

###### This is an angular project version 13

`Installation process`

- Go to [Repo Web](https://dev.azure.com/valeyo/Valeyo.Insurance/_git/Securian.Insurance.Web) and clone it in your local machine
- In the terminal open the project folder where you just cloned it
- If you are running first time, then in the terminal folder run the command `npm install`
- If you already ran it once, ignore previous step and instead run `npm run`
- http://localhost:4200/ is our local url to test the project
- Then move to alpha branch which is the branch we are using to develop new features
- From there you can create your own branch to start making you new changes
- When everything gets ready you have to create a pull request from your local branch to alpha brach, and a pipeline will run automatically to build, create, and deploy the project to [DEV](https://dev-securianinsuux-alpha.ori.valeyo.com/) environment
